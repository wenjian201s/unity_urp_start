# 从零到完整落地实现镜头光晕

本文基于当前目录 `Assets/镜头光晕` 的实现编写。当前项目没有使用 Unity 内置 Lens Flare 组件，而是自己实现了一套屏幕空间光晕系统：光源通过 `MFFlareLauncher` 注册到主相机，主相机上的 `MFLensFlare` 负责计算光源屏幕坐标、读取深度图判断遮挡、驱动淡入淡出、动态生成多个 Quad 组成的 Mesh，最后用自定义透明叠加 Shader 绘制到相机前方。

## 项目文件如何组成一条渲染链路

这套系统的文件可以按职责分成数据、运行时、GPU、编辑器和示例资源。数据文件描述一条光晕由哪些小图块组成，运行时代码决定这些图块何时出现以及摆在屏幕哪里，GPU 代码负责用深度图判断光源是否被遮挡，编辑器脚本让开发者可以直接在 Inspector 里切图、选图和调参数。


当前示例场景 `镜头光晕.unity` 中，`Main Camera` 带有 `MFLensFlare`，它引用 `Moonflow_Lensflare.mat` 和 `LightOcclusionChecker.compute`，`fadeoutTime` 为 `0.2`，`DebugMode` 为开启状态。`Directional Light` 带有 `MFFlareLauncher`，设置为方向光模式，引用 `FlareAsset.asset`。URP 管线资源 `光晕.asset` 开启了深度纹理，这一点决定了 Compute Shader 是否能拿到 `_CameraDepthTexture`。

## 从零实现时先定义数据，而不是先写渲染

镜头光晕不是一张单独图片。当前实现把一条 Lens Flare 拆成多个 `MFFlareSpriteData`，每个元素保存贴图图集中的采样矩形、颜色、缩放、偏移位置、是否旋转、是否受光源颜色影响。`MFFlareAsset` 是 ScriptableObject，它把同一组光晕配置保存成 Unity 资源，让多个光源可以复用同一份数据。

`offset` 是这套系统的核心参数。程序先计算光源屏幕坐标相对屏幕中心的向量，再用 `offset` 缩放这个向量，最后把结果加回屏幕中心。`offset = 0` 时元素靠近屏幕中心，正值会沿光源偏移方向扩展，负值会跑到另一侧，从而形成典型的鬼影光斑队列。

```csharp
using System;
using UnityEngine;

[Serializable]
public struct MFFlareSpriteData
{
    public float useLightColor;
    public bool useRotation;
    public int index;
    public Rect block;
    public float scale;

    [Range(-1.5f, 1)]
    public float offset;

    [ColorUsage(true, true)]
    public Color color;
}
```

`block` 使用归一化 UV 矩形，而不是像素矩形。这样运行时不需要知道具体 Sprite 子资源，只要用图集尺寸和 UV 比例就能算出 Quad 在屏幕上的像素大小。当前项目的 `FlareAsset.asset` 使用 `Flare50mm.png`，开启 `fadeWithScale` 和 `fadeWithAlpha`，一共配置了 8 个光晕元素，其中一个大条纹元素开启了旋转，用来随着光源和屏幕中心的方向转动。

```csharp
using System.Collections.Generic;
using UnityEngine;

public class MFFlareAsset : ScriptableObject
{
    public Texture2D flareSprite;
    public bool fadeWithScale;
    public bool fadeWithAlpha;
    public List<MFFlareSpriteData> spriteBlocks;
}
```

当前项目支持两种创建光晕资源的方式。`MFFlareAssetCell` 适合规则网格图集，开发者输入列数和行数以后，编辑器按格子计算 UV。`MFFlareAssetSlicer` 适合已经用 Unity Sprite Editor 切好的 Multiple Sprite 图集，编辑器读取贴图子资源来生成 `spriteBlocks`。

## 光源用 Launcher 注册到主相机

`MFFlareLauncher` 挂在真实的 `Light` 物体上。它的职责很窄：缓存 `Light`，找到 `Camera.main`，从主相机上获取 `MFLensFlare`，然后在启用时注册、禁用时反注册。这样主控制器不用每帧扫描场景里所有 Light，也不会在光源被禁用后继续维护无效数据。

方向光没有真实的世界位置，代码用 `directionalLight` 告诉主控制器把它当成一个沿光照方向放在远处的虚拟光源。`useLightIntensity` 控制最终顶点色是否乘以 `Light.intensity`，当前示例方向光关闭了这个选项，所以光晕亮度主要由 `MFFlareSpriteData.color` 决定。

```csharp
using UnityEngine;

[RequireComponent(typeof(Light))]
public class MFFlareLauncher : MonoBehaviour
{
    public bool directionalLight;
    public bool useLightIntensity;
    public MFFlareAsset asset;

    [HideInInspector]
    public Light lightSource;

    private Camera _mainCam;

    private void OnEnable()
    {
        lightSource = GetComponent<Light>();
        _mainCam = Camera.main;

        if (_mainCam == null)
        {
            Debug.LogWarning("MFFlareLauncher 找不到 MainCamera，请确认场景中有一个 Tag 为 MainCamera 的相机。", this);
            return;
        }

        MFLensFlare lensFlare = _mainCam.GetComponent<MFLensFlare>();
        if (lensFlare == null)
        {
            Debug.LogWarning("MFFlareLauncher 找不到 MFLensFlare，请把 MFLensFlare 脚本挂到 MainCamera 上。", this);
            return;
        }

        lensFlare.AddLight(this);
    }

    private void Reset()
    {
        lightSource = GetComponent<Light>();
    }

    private void OnDisable()
    {
        if (_mainCam == null)
        {
            return;
        }

        MFLensFlare lensFlare = _mainCam.GetComponent<MFLensFlare>();
        if (lensFlare == null)
        {
            return;
        }

        lensFlare.RemoveLight(this);
    }
}
```

## 主控制器的状态、缓存和生命周期

`MFLensFlare` 是运行时核心。它维护两个字典：`_flareDict` 保存所有已注册光源，`_activeFlareDict` 保存当前占用遮挡检测槽位的光源。遮挡检测固定使用 8 个槽位，对应两个 `ComputeBuffer`，一个保存屏幕像素坐标，一个保存 Compute Shader 读回的线性深度。固定槽位让 GPU 数据结构简单，但也意味着同一帧最多只有 8 个活跃光源参与遮挡检测。

每个光源对应一个 `FlareStatusData`。这里保存源光屏幕坐标、每个光斑的屏幕中心、淡入淡出强度、可见性、Mesh 顶点数组、UV 数组、顶点色数组、三角形索引和遮挡检测槽位。初始化时就把 UV 和三角形索引写好，因为它们只和 `MFFlareAsset.spriteBlocks` 有关，每帧变化的主要是顶点位置和顶点颜色。

```csharp
public enum FadeState
{
    Render = 0,
    FadeIn = 1,
    FadeOut = 2,
    Unrendered = 3
}

public class FlareStatusData
{
    public Vector3 sourceCoordinate;
    public Vector3[] flareWorldPosCenter;
    public float flareScale;
    public bool isInScreen;
    public bool isVisible;
    public FadeState fadeState;
    public Vector4 sourceScreenPos;
    public Mesh flareMesh;
    public Vector3[] vertices;
    public Vector2[] uv;
    public Color[] vertColor;
    public int[] triangle;
    public int srcIndex;
}
```

完整的 `MFLensFlare` 实现如下。代码运行在 URP 的相机渲染后回调中准备遮挡数据，在 `Update` 中处理状态机和绘制。它没有插入自定义 RenderPass，而是通过 `Graphics.DrawMesh` 把动态 Mesh 提交给当前相机，这使实现较直接，也要求材质本身设置为透明叠加并关闭深度测试。

```csharp
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using System;
using UnityEngine.Rendering;

#if UNITY_EDITOR
using UnityEditor;
#endif

public class MFLensFlare : MonoBehaviour
{
    private static readonly int PIPELINE_DEPTH_TEX = Shader.PropertyToID("_CameraDepthTexture");
    private static readonly int Z_BUFFER_PARAMS = Shader.PropertyToID("_ZBufferParams");
    private static readonly string CAMERA_COMPARE_TAG = "MainCamera";

    public bool DebugMode;

    [Space(10)]
    public Material material;
    public float fadeoutTime = 0.25f;
    public ComputeShader cs_PrepareLightOcclusion;

    private Dictionary<MFFlareLauncher, FlareStatusData> _flareDict;
    private Dictionary<MFFlareLauncher, FlareStatusData> _activeFlareDict;
    private Camera _camera;
    private Vector2 _halfScreen;
    private MaterialPropertyBlock _propertyBlock;
    private Vector3 _screenCenter;
    private Queue<Mesh> _meshPool;

    private static readonly int STATIC_FLARESCREENPOS = Shader.PropertyToID("_FlareScreenPos");
    private static readonly int STATIC_BaseMap = Shader.PropertyToID("_MainTex");
    private static readonly float DISTANCE = 1f;

    private int _csKernel;
    private ComputeBuffer _cbLightOcclusionCheckBuffer;
    private ComputeBuffer _cbLightUVBuffer;

    public struct ScreenSpaceUV
    {
        public int x;
        public int y;
    }

    private ScreenSpaceUV[] _screenSpaceLightSrcUV;
    private float[] _lightSourceDepth =
    {
        1.0f, 1.0f, 1.0f, 1.0f,
        1.0f, 1.0f, 1.0f, 1.0f
    };

    private Queue<int> _emptyIndex = new Queue<int>();

    private static readonly int CS_LIGHT_UV = Shader.PropertyToID("_LightUV");
    private static readonly int CS_IS_LIGHT_OCCLUDED = Shader.PropertyToID("_LightSourceDepth");
    private static readonly int CS_DEPTHTEX_NAME = Shader.PropertyToID("_DepthTex");
    private static readonly int Z_BUFFER_DELIVERED = Shader.PropertyToID("_ZBufferDelivered");

#if UNITY_EDITOR
    public Dictionary<MFFlareLauncher, FlareStatusData> FlareDict => _flareDict;
    public Dictionary<MFFlareLauncher, FlareStatusData> ActiveDict => _activeFlareDict;
    public float[] Editor_LightSrcDepth => _lightSourceDepth;
#endif

    private void Awake()
    {
        _camera = GetComponent<Camera>();
        _propertyBlock = new MaterialPropertyBlock();
        _flareDict = new Dictionary<MFFlareLauncher, FlareStatusData>();
        _activeFlareDict = new Dictionary<MFFlareLauncher, FlareStatusData>();
        _meshPool = new Queue<Mesh>();

        if (cs_PrepareLightOcclusion != null)
        {
            _csKernel = cs_PrepareLightOcclusion.FindKernel("PrepareLightOcclusion");
        }

        _cbLightOcclusionCheckBuffer = new ComputeBuffer(8, sizeof(float), ComputeBufferType.Structured);
        _cbLightUVBuffer = new ComputeBuffer(8, sizeof(int) * 2, ComputeBufferType.Structured);
        RenderPipelineManager.endCameraRendering += AddRenderPass;
        _screenSpaceLightSrcUV = new ScreenSpaceUV[8];

        for (int i = 0; i < 8; i++)
        {
            _emptyIndex.Enqueue(i);
            _screenSpaceLightSrcUV[i] = new ScreenSpaceUV { x = 0, y = 0 };
        }
    }

    private void OnDisable()
    {
        RenderPipelineManager.endCameraRendering -= AddRenderPass;
        _cbLightOcclusionCheckBuffer?.Release();
        _cbLightUVBuffer?.Release();
    }

    private FlareStatusData InitFlareData(MFFlareLauncher mfFlareLauncher)
    {
        int flareCount = mfFlareLauncher.asset.spriteBlocks.Count;
        FlareStatusData statusData = new FlareStatusData
        {
            sourceCoordinate = Vector3.zero,
            flareWorldPosCenter = new Vector3[flareCount],
            flareScale = 0,
            fadeState = FadeState.Render,
            sourceScreenPos = Vector4.zero,
            flareMesh = _meshPool.Count > 0 ? _meshPool.Dequeue() : new Mesh(),
            vertices = new Vector3[flareCount * 4],
            triangle = new int[flareCount * 6],
            uv = new Vector2[flareCount * 4],
            vertColor = new Color[flareCount * 4],
            srcIndex = -1
        };

        for (int i = 0; i < flareCount; i++)
        {
            Rect rect = mfFlareLauncher.asset.spriteBlocks[i].block;
            statusData.uv[i * 4] = rect.position;
            statusData.uv[i * 4 + 1] = rect.position + new Vector2(rect.width, 0);
            statusData.uv[i * 4 + 2] = rect.position + new Vector2(0, rect.height);
            statusData.uv[i * 4 + 3] = rect.position + rect.size;

            statusData.triangle[i * 6] = i * 4;
            statusData.triangle[i * 6 + 1] = i * 4 + 3;
            statusData.triangle[i * 6 + 2] = i * 4 + 1;
            statusData.triangle[i * 6 + 3] = i * 4;
            statusData.triangle[i * 6 + 4] = i * 4 + 2;
            statusData.triangle[i * 6 + 5] = i * 4 + 3;
        }

        return statusData;
    }

    public void AddLight(MFFlareLauncher mfFlareLauncher)
    {
        if (mfFlareLauncher == null || mfFlareLauncher.asset == null)
        {
            return;
        }

        if (_flareDict.ContainsKey(mfFlareLauncher))
        {
            return;
        }

        if (DebugMode)
        {
            Debug.Log("Add Light " + mfFlareLauncher.gameObject.name + " to FlareList");
        }

        FlareStatusData flareData = InitFlareData(mfFlareLauncher);
        _flareDict.Add(mfFlareLauncher, flareData);
        ActiveLight(mfFlareLauncher, ref flareData);
    }

    public void ActiveLight(MFFlareLauncher mfFlareLauncher, ref FlareStatusData flareData)
    {
        if (_activeFlareDict.ContainsKey(mfFlareLauncher))
        {
            return;
        }

        if (_emptyIndex.Count <= 0)
        {
            if (DebugMode)
            {
                Debug.LogWarning("MFLensFlare active light limit reached. Max active lights = 8.");
            }

            flareData.srcIndex = -1;
            return;
        }

        if (DebugMode)
        {
            Debug.Log("Active Light " + mfFlareLauncher.gameObject.name + " to ActiveList");
        }

        flareData.srcIndex = _emptyIndex.Dequeue();
        _activeFlareDict.Add(mfFlareLauncher, flareData);
    }

    public void RemoveLight(MFFlareLauncher mfFlareLauncher)
    {
        if (mfFlareLauncher == null)
        {
            return;
        }

        InactiveLight(mfFlareLauncher);

        if (DebugMode)
        {
            Debug.Log("Remove Light " + mfFlareLauncher.gameObject.name + " from FlareList");
        }

        if (_flareDict.TryGetValue(mfFlareLauncher, out FlareStatusData flareState))
        {
            if (flareState.flareMesh != null)
            {
                flareState.flareMesh.Clear();
                _meshPool.Enqueue(flareState.flareMesh);
            }

            _flareDict.Remove(mfFlareLauncher);
        }
    }

    private void InactiveLight(MFFlareLauncher mfFlareLauncher)
    {
        if (mfFlareLauncher == null)
        {
            return;
        }

        if (_activeFlareDict.TryGetValue(mfFlareLauncher, out FlareStatusData flareState))
        {
            if (flareState.srcIndex >= 0)
            {
                _emptyIndex.Enqueue(flareState.srcIndex);
            }

            flareState.srcIndex = -1;
            _activeFlareDict.Remove(mfFlareLauncher);

            if (DebugMode)
            {
                Debug.Log("Inactive Light " + mfFlareLauncher.gameObject.name + " from FlareList");
            }
        }
    }

    private void Update()
    {
        if (_camera == null)
        {
            return;
        }

        _halfScreen = new Vector2(
            _camera.scaledPixelWidth / 2f + _camera.pixelRect.xMin,
            _camera.scaledPixelHeight / 2f + _camera.pixelRect.yMin
        );

        Transform cameraTransform = _camera.transform;
        _screenCenter = cameraTransform.position + cameraTransform.forward * 0.1f;

        foreach (var pair in _flareDict)
        {
            FlareStatusData flareStatusData = pair.Value;
            MFFlareLauncher lightSource = pair.Key;

            GetSourceCoordinate(lightSource, ref flareStatusData);
            CheckIn(lightSource, ref flareStatusData);

            if (flareStatusData.flareScale > 0)
            {
                if (flareStatusData.flareScale >= 1)
                {
                    flareStatusData.fadeState = flareStatusData.isVisible ? FadeState.Render : FadeState.FadeOut;
                }
                else
                {
                    flareStatusData.fadeState = flareStatusData.isVisible ? FadeState.FadeIn : FadeState.FadeOut;
                }
            }
            else
            {
                if (!flareStatusData.isInScreen)
                {
                    flareStatusData.fadeState = FadeState.Unrendered;
                }
                else
                {
                    if (!_activeFlareDict.TryGetValue(lightSource, out _))
                    {
                        ActiveLight(lightSource, ref flareStatusData);
                    }

                    flareStatusData.fadeState = FadeState.FadeIn;
                }
            }

            if (flareStatusData.fadeState != FadeState.Unrendered)
            {
                CalculateMeshData(lightSource, ref flareStatusData);
            }

            float fadeTime = Mathf.Max(0.0001f, fadeoutTime);
            switch (flareStatusData.fadeState)
            {
                case FadeState.FadeIn:
                    flareStatusData.flareScale += Time.deltaTime / fadeTime;
                    flareStatusData.flareScale = Mathf.Clamp01(flareStatusData.flareScale);
                    break;

                case FadeState.FadeOut:
                    flareStatusData.flareScale -= Time.deltaTime / fadeTime;
                    flareStatusData.flareScale = Mathf.Clamp01(flareStatusData.flareScale);
                    break;

                case FadeState.Unrendered:
                    flareStatusData.flareScale = 0;
                    InactiveLight(lightSource);
                    break;

                case FadeState.Render:
                    flareStatusData.flareScale = 1;
                    break;
            }

            CreateMesh(lightSource, ref flareStatusData);

            if (DebugMode)
            {
                DebugDrawMeshPos(lightSource, flareStatusData);
            }
        }
    }

    private void GetSourceCoordinate(MFFlareLauncher lightSource, ref FlareStatusData statusData)
    {
        Transform lightSourceTransform = lightSource.transform;
        Vector3 sourceWorldPosition = lightSource.directionalLight
            ? _camera.transform.position - lightSourceTransform.forward * 10000f
            : lightSourceTransform.position;

        Vector3 sourceScreenPos = _camera.WorldToScreenPoint(sourceWorldPosition);
        statusData.sourceCoordinate = sourceScreenPos;
    }

    private void CheckIn(MFFlareLauncher lightSource, ref FlareStatusData statusData)
    {
        bool outOfScreen =
            statusData.sourceCoordinate.x < _camera.pixelRect.xMin ||
            statusData.sourceCoordinate.y < _camera.pixelRect.yMin ||
            statusData.sourceCoordinate.x > _camera.pixelRect.xMax ||
            statusData.sourceCoordinate.y > _camera.pixelRect.yMax;

        Vector3 lightDirection = lightSource.directionalLight
            ? Vector3.Normalize(_camera.transform.position - lightSource.transform.forward * 10000f)
            : Vector3.Normalize(lightSource.transform.position - _camera.transform.position);

        bool behindCamera = Vector3.Dot(lightDirection, _camera.transform.forward) < 0.25f;

        if (outOfScreen || behindCamera)
        {
            statusData.sourceScreenPos = Vector4.zero;
            statusData.isVisible = false;
            statusData.isInScreen = false;
            return;
        }

        statusData.isInScreen = true;

        if (statusData.srcIndex == -1)
        {
            statusData.isVisible = false;
            return;
        }

        Vector4 screenUV = statusData.sourceCoordinate;
        screenUV.x /= _camera.pixelWidth;
        screenUV.y /= _camera.pixelHeight;
        screenUV.w = lightSource.directionalLight ? 1 : 0;
        statusData.sourceScreenPos = screenUV;

        float worldSpaceDepth = _lightSourceDepth[statusData.srcIndex];

        if (lightSource.directionalLight)
        {
            if (worldSpaceDepth >= _camera.farClipPlane - _camera.nearClipPlane)
            {
                statusData.isVisible = true;
            }
            else
            {
                statusData.sourceScreenPos = Vector4.zero;
                statusData.isVisible = false;
            }
        }
        else
        {
            float lightDistance =
                Vector3.Magnitude(lightSource.transform.position - _camera.transform.position) -
                _camera.nearClipPlane;

            if (worldSpaceDepth > lightDistance)
            {
                statusData.isVisible = true;
            }
            else
            {
                statusData.sourceScreenPos = Vector4.zero;
                statusData.isVisible = false;
            }
        }
    }

    private void PrepareLightOcclusion()
    {
        if (cs_PrepareLightOcclusion == null)
        {
            return;
        }

        DeliverUV();
        Texture depthTex = Shader.GetGlobalTexture(PIPELINE_DEPTH_TEX);

        if (depthTex == null)
        {
            return;
        }

        cs_PrepareLightOcclusion.SetTexture(_csKernel, CS_DEPTHTEX_NAME, depthTex);
        _cbLightOcclusionCheckBuffer.SetData(_lightSourceDepth);
        _cbLightUVBuffer.SetData(_screenSpaceLightSrcUV);
        cs_PrepareLightOcclusion.SetBuffer(_csKernel, CS_IS_LIGHT_OCCLUDED, _cbLightOcclusionCheckBuffer);
        cs_PrepareLightOcclusion.SetBuffer(_csKernel, CS_LIGHT_UV, _cbLightUVBuffer);

        Vector4 zBufferParam = Shader.GetGlobalVector(Z_BUFFER_PARAMS);
        cs_PrepareLightOcclusion.SetVector(Z_BUFFER_DELIVERED, zBufferParam);
        cs_PrepareLightOcclusion.Dispatch(_csKernel, 1, 1, 1);
        _cbLightOcclusionCheckBuffer.GetData(_lightSourceDepth);
    }

    private void DeliverUV()
    {
        foreach (var pair in _flareDict)
        {
            FlareStatusData src = pair.Value;

            if (src.srcIndex != -1)
            {
                _screenSpaceLightSrcUV[src.srcIndex].x =
                    Mathf.Clamp((int)src.sourceCoordinate.x, 0, _camera.pixelWidth - 1);

                _screenSpaceLightSrcUV[src.srcIndex].y =
                    Mathf.Clamp((int)src.sourceCoordinate.y, 0, _camera.pixelHeight - 1);
            }
        }
    }

    private void AddRenderPass(ScriptableRenderContext context, Camera camera)
    {
        if (camera != null && camera.gameObject.CompareTag(CAMERA_COMPARE_TAG))
        {
            PrepareLightOcclusion();
        }
    }

    private void CalculateMeshData(MFFlareLauncher lightSource, ref FlareStatusData statusData)
    {
        int flareCount = lightSource.asset.spriteBlocks.Count;
        Vector3[] oneFlareLine = new Vector3[flareCount];

        for (int i = 0; i < flareCount; i++)
        {
            Vector2 sourceOffsetFromCenter = new Vector2(
                statusData.sourceCoordinate.x - _halfScreen.x,
                statusData.sourceCoordinate.y - _halfScreen.y
            );

            Vector2 realOffset = sourceOffsetFromCenter * lightSource.asset.spriteBlocks[i].offset;
            oneFlareLine[i] = new Vector3(_halfScreen.x + realOffset.x, _halfScreen.y + realOffset.y, DISTANCE);
        }

        statusData.flareWorldPosCenter = oneFlareLine;
    }

    private void CreateMesh(MFFlareLauncher lightSource, ref FlareStatusData statusData)
    {
        if (statusData.flareScale <= 0)
        {
            return;
        }

        if (lightSource.asset == null || lightSource.asset.flareSprite == null || material == null)
        {
            return;
        }

        Light source = lightSource.GetComponent<Light>();
        if (source == null)
        {
            return;
        }

        Texture2D tex = lightSource.asset.flareSprite;
        float angle =
            (45f + Vector2.SignedAngle(
                Vector2.up,
                new Vector2(
                    statusData.sourceCoordinate.x - _halfScreen.x,
                    statusData.sourceCoordinate.y - _halfScreen.y
                )
            )) / 180f * Mathf.PI;

        int flareCount = lightSource.asset.spriteBlocks.Count;

        for (int i = 0; i < flareCount; i++)
        {
            MFFlareSpriteData spriteData = lightSource.asset.spriteBlocks[i];
            Rect rect = spriteData.block;

            float scaleFade = lightSource.asset.fadeWithScale
                ? statusData.flareScale * 0.5f + 0.5f
                : 1f;

            Vector2 halfSize = new Vector2(
                tex.width * rect.width * 0.5f * spriteData.scale * scaleFade,
                tex.height * rect.height * 0.5f * spriteData.scale * scaleFade
            );

            Vector3 flarePos = statusData.flareWorldPosCenter[i];

            if (spriteData.useRotation)
            {
                float magnitude = Mathf.Sqrt(halfSize.x * halfSize.x + halfSize.y * halfSize.y);
                float cos = magnitude * Mathf.Cos(angle);
                float sin = magnitude * Mathf.Sin(angle);

                statusData.vertices[i * 4] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x - sin, flarePos.y + cos, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 1] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x - cos, flarePos.y - sin, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 2] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x + cos, flarePos.y + sin, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 3] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x + sin, flarePos.y - cos, flarePos.z)) -
                    _screenCenter;
            }
            else
            {
                statusData.vertices[i * 4] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x - halfSize.x, flarePos.y + halfSize.y, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 1] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x - halfSize.x, flarePos.y - halfSize.y, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 2] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x + halfSize.x, flarePos.y + halfSize.y, flarePos.z)) -
                    _screenCenter;

                statusData.vertices[i * 4 + 3] =
                    _camera.ScreenToWorldPoint(new Vector3(flarePos.x + halfSize.x, flarePos.y - halfSize.y, flarePos.z)) -
                    _screenCenter;
            }

            Color vertexAddColor = spriteData.color;
            Color lightColor = Color.white;
            lightColor.r = Mathf.Lerp(1f, source.color.r, spriteData.useLightColor);
            lightColor.g = Mathf.Lerp(1f, source.color.g, spriteData.useLightColor);
            lightColor.b = Mathf.Lerp(1f, source.color.b, spriteData.useLightColor);
            lightColor.a = 1f;
            lightColor *= lightSource.useLightIntensity ? source.intensity : 1f;

            float offsetAlpha = (1.5f - Mathf.Abs(spriteData.offset)) / 1.5f;
            float screenCenterAlpha =
                1f -
                Mathf.Min(
                    1f,
                    new Vector2(flarePos.x - _halfScreen.x, flarePos.y - _halfScreen.y).magnitude /
                    new Vector2(_halfScreen.x, _halfScreen.y).magnitude
                );

            float fadeAlpha = lightSource.asset.fadeWithAlpha ? statusData.flareScale : 1f;
            vertexAddColor *= new Vector4(lightColor.r, lightColor.g, lightColor.b, offsetAlpha * screenCenterAlpha) *
                              fadeAlpha;
            vertexAddColor = vertexAddColor.linear;

            statusData.vertColor[i * 4] = vertexAddColor;
            statusData.vertColor[i * 4 + 1] = vertexAddColor;
            statusData.vertColor[i * 4 + 2] = vertexAddColor;
            statusData.vertColor[i * 4 + 3] = vertexAddColor;
        }

        statusData.flareMesh.Clear();
        statusData.flareMesh.vertices = statusData.vertices;
        statusData.flareMesh.uv = statusData.uv;
        statusData.flareMesh.triangles = statusData.triangle;
        statusData.flareMesh.colors = statusData.vertColor;

        _propertyBlock.SetTexture(STATIC_BaseMap, lightSource.asset.flareSprite);
        _propertyBlock.SetVector(STATIC_FLARESCREENPOS, statusData.sourceScreenPos);
        Graphics.DrawMesh(statusData.flareMesh, _screenCenter, Quaternion.identity, material, 0, _camera, 0, _propertyBlock);
    }

    private void DebugDrawMeshPos(MFFlareLauncher lightSource, FlareStatusData statusData)
    {
        if (statusData.flareWorldPosCenter == null)
        {
            return;
        }

        for (int i = 0; i < lightSource.asset.spriteBlocks.Count; i++)
        {
            Debug.DrawLine(_camera.transform.position, _camera.ScreenToWorldPoint(statusData.flareWorldPosCenter[i]));
        }
    }

    private void OnDestroy()
    {
        if (_flareDict != null)
        {
            foreach (var pair in _flareDict)
            {
                if (pair.Value.flareMesh != null)
                {
                    _meshPool.Enqueue(pair.Value.flareMesh);
                }
            }
        }

        if (_meshPool != null)
        {
            while (_meshPool.Count > 0)
            {
                Destroy(_meshPool.Dequeue());
            }

            _meshPool.Clear();
        }
    }
}
```

`Update` 的状态机负责避免光晕突然出现或消失。光源可见时，`flareScale` 向 1 增加；光源被遮挡或离开屏幕时，`flareScale` 向 0 减少；完全不可见且不在屏幕内时，状态变成 `Unrendered`，并归还 ComputeBuffer 槽位。`fadeoutTime` 使用 `Mathf.Max(0.0001f, fadeoutTime)` 防止除零，这让 Inspector 里误填 0 时也不会炸掉更新循环。

遮挡判断发生在 `CheckIn`。普通点光源或聚光灯比较深度图线性深度和光源到相机的线性距离，如果深度图中的最近物体比光源更远，就认为光源可见。方向光没有距离，当前实现把它投到远处，然后要求深度图读数接近远裁剪距离；这意味着方向光光晕通常只会在采样点看到天空或远处空背景时出现。

## Compute Shader 负责把深度图变成可比较的线性深度

URP 的 `_CameraDepthTexture` 存的是投影后的非线性深度，不能直接和世界距离比较。`LightOcclusionChecker.compute` 根据 CPU 上传的像素坐标读取深度图，再用 `_ZBufferParams` 的等价参数执行线性化公式 `1.0 / (z * depth + w)`，最后把结果写入 `_LightSourceDepth`。C# 侧在下一步用 `GetData` 把这 8 个 float 读回 CPU。

```hlsl
#pragma kernel PrepareLightOcclusion

struct LightUV
{
    int x;
    int y;
};

RWStructuredBuffer<float> _LightSourceDepth;
RWStructuredBuffer<LightUV> _LightUV;
Texture2D<float4> _DepthTex;
float4 _ZBufferDelivered;

float LinearEyeDepth(float depth, float4 zBufferParam)
{
    return 1.0 / (zBufferParam.z * depth + zBufferParam.w);
}

[numthreads(8, 1, 1)]
void PrepareLightOcclusion(uint3 id : SV_DispatchThreadID)
{
    _LightSourceDepth[id.x] =
        LinearEyeDepth(
            _DepthTex[
                uint2(
                    _LightUV[id.x].x,
                    _LightUV[id.x].y
                )
            ].x,
            _ZBufferDelivered
        );
}
```

当前 Dispatch 固定为 `(1, 1, 1)`，配合 `[numthreads(8, 1, 1)]` 正好处理 8 个槽位。这个设计和 `_lightSourceDepth`、`_screenSpaceLightSrcUV` 的数组长度一致。如果未来要支持更多光源，需要同时扩大数组、ComputeBuffer、空闲槽位队列，并按光源数量计算 Dispatch 组数。

## Shader 只负责透明叠加和图集采样

当前材质 `Moonflow_Lensflare.mat` 使用 `Moonflow/Lensflare` Shader，主贴图绑定到 `Flare50mm.png`。实际绘制时 `MFLensFlare` 会通过 `MaterialPropertyBlock` 给每个光源覆盖 `_MainTex` 和 `_FlareScreenPos`，因此多个光源可以共享一个材质实例而不产生材质拷贝。

`Blend One One` 是典型加法混合，光晕颜色会加到当前画面上。`ZWrite Off` 避免写入深度，`ZTest Off` 保证光晕 Mesh 不被自己的深度测试挡掉。遮挡已经由 Compute Shader 提前决定，所以 Shader 里不再采样深度图。

```hlsl
Shader "Moonflow/Lensflare"
{
    Properties
    {
        _MainTex ("Texture", 2D) = "white" {}
    }

    SubShader
    {
        Tags
        {
            "RenderType" = "Transparent"
            "IgnoreProjector" = "True"
            "PreviewType" = "Plane"
            "PerformanceChecks" = "False"
            "RenderPipeline" = "UniversalPipeline"
        }

        LOD 100

        Pass
        {
            Blend One One
            ZWrite Off
            ZTest Off

            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/core.hlsl"

            struct appdata
            {
                float4 vertex : POSITION;
                float2 uv : TEXCOORD0;
                half4 color : COLOR;
            };

            struct v2f
            {
                float4 vertex : SV_POSITION;
                float2 uv : TEXCOORD0;
                half4 color : TEXCOORD1;
            };

            Texture2D _MainTex;
            SamplerState sampler_MainTex;
            float4 _MainTex_ST;
            half4 _FlareScreenPos;

            v2f vert(appdata v)
            {
                v2f o;
                o.vertex = TransformObjectToHClip(v.vertex);
                o.uv.xy = v.uv;
                o.color = v.color;
                return o;
            }

            half4 frag(v2f i) : SV_Target
            {
                half4 col = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, i.uv.xy) * i.color;
                return col;
            }
            ENDHLSL
        }
    }
}
```

## 动态 Mesh 是屏幕空间 Quad 的批处理容器

每个光晕元素最终都是一个 Quad。`InitFlareData` 为每个元素预写 4 个 UV 和 6 个三角形索引，`CreateMesh` 每帧根据光斑屏幕中心、贴图块尺寸、缩放、旋转和淡入淡出更新顶点。顶点先用屏幕坐标算出，再通过 `Camera.ScreenToWorldPoint` 转成相机前方的世界坐标，并减去 `_screenCenter` 作为局部顶点。最后调用 `Graphics.DrawMesh(mesh, _screenCenter, Quaternion.identity, material, 0, _camera, 0, block)` 绘制。

颜色合成在 CPU 上完成。基础颜色来自 `MFFlareSpriteData.color`，光源颜色按 `useLightColor` 插值到白色和 `Light.color` 之间，`useLightIntensity` 决定是否乘以光源强度。透明度还会乘以两个位置衰减：`offsetAlpha` 让过远的鬼影更弱，`screenCenterAlpha` 让靠屏幕边缘的光斑更暗。最后如果 `fadeWithAlpha` 开启，顶点色再乘以 `flareScale`。

## Cell 编辑器用于规则图集

`MFFlareCellAssetEditor` 给 `MFFlareAssetCell` 提供自定义 Inspector。它用 `modelCell.x` 和 `modelCell.y` 绘制一个可点击网格，每个格子对应图集中的一个 UV 区域。点击格子会向 `spriteBlocks` 添加一条 `MFFlareSpriteData`，下方 Inspector 可以继续调 `index`、`useRotation`、`useLightColor`、`offset`、`color` 和 `scale`。

```csharp
using System.Collections.Generic;
using UnityEditor;
using UnityEngine;

[CustomEditor(typeof(MFFlareAssetCell))]
public class MFFlareCellAssetEditor : Editor

        return new Rect(
```

这个编辑器当前的 `GetFlareRect` 在计算行号时使用了 `index / modelCell.y`。当图集是 4x4 这种正方形网格时不会暴露问题；如果要支持 4x2 或 8x4 这类非正方形网格，行号通常应使用 `index / modelCell.x`，否则 UV 的 y 坐标会错位。

`MFFlareFreeAssetEditor` 给 `MFFlareAssetSlicer` 提供自定义 Inspector。它读取 `flareSprite` 路径下的所有 Sprite 子资源，把每个切片画成按钮。点击按钮后添加一条 `MFFlareSpriteData`，后续每次绘制时根据当前 `index` 把 `Sprite.rect` 转换成归一化 UV Rect。

这里使用 `AssetDatabase.FindAssets(_tmp.name)[0]` 查找贴图路径。如果项目中存在同名资源，可能会读到错误路径。更稳定的写法是 `AssetDatabase.GetAssetPath(_tmp)`，因为 ObjectField 已经拿到了准确的 `Texture2D` 对象。

## 自定义 Inspector 显示运行时遮挡调试信息

`MFLensFlare.cs` 末尾还包含一个编辑器调试面板。它显示注册光源数量、活跃光源、是否方向光、是否在屏幕内、是否可见、ComputeBuffer 槽位、深度图读数和光源到相机距离。这个面板只在 `UNITY_EDITOR` 下编译，不会进游戏包。

```csharp
#if UNITY_EDITOR
[CustomEditor(typeof(MFLensFlare))]
public class MFLensflareEditor : Editor
{
    public MFLensFlare _target;

    private void OnEnable()
    {
        _target = target as MFLensFlare;
    }

    public override void OnInspectorGUI()
    {
        if (_target.FlareDict != null && _target.FlareDict.Count != 0)
        {
            EditorGUILayout.LabelField("Light Count: ", _target.FlareDict.Count.ToString());
        }

        EditorGUILayout.Space();

        if (_target.ActiveDict != null && _target.ActiveDict.Count != 0)
        {
            EditorGUIUtility.fieldWidth = 50;

            using (new EditorGUILayout.HorizontalScope())
            {
                EditorGUILayout.TextField("Dir");
                EditorGUILayout.TextField("InScreen");
                EditorGUILayout.TextField("Visible");
                EditorGUILayout.TextField("SrcIndex");
                EditorGUILayout.TextField("DepthOnTex");
                EditorGUILayout.TextField("LinearDist");
            }

            foreach (var pair in _target.ActiveDict)
            {
                using (new EditorGUILayout.HorizontalScope("box"))
                {
                    EditorGUILayout.TextField(pair.Key.directionalLight.ToString());
                    EditorGUILayout.TextField(pair.Value.isInScreen.ToString());
                    EditorGUILayout.TextField(pair.Value.isVisible.ToString());
                    EditorGUILayout.TextField(pair.Value.srcIndex.ToString());

                    float oDepth = 0f;
                    if (pair.Value.srcIndex >= 0 &&
                        _target.Editor_LightSrcDepth != null &&
                        pair.Value.srcIndex < _target.Editor_LightSrcDepth.Length)
                    {
                        oDepth = _target.Editor_LightSrcDepth[pair.Value.srcIndex];
                    }

                    EditorGUILayout.TextField(oDepth.ToString());

                    float linearDist = 0f;
                    if (Camera.main != null)
                    {
                        linearDist = Vector3.Magnitude(pair.Key.transform.position - Camera.main.transform.position);
                    }

                    EditorGUILayout.TextField(linearDist.ToString());
                }
            }
        }

        EditorGUILayout.Space();
        base.OnInspectorGUI();
    }
}
#endif
```

## 在当前项目中从零搭建这套系统

从空场景开始搭建时，先确保项目使用 URP，并在管线资源中开启深度纹理。当前 `光晕.asset` 已经开启 `m_RequireDepthTexture: 1`，相机的 URP Additional Camera Data 使用管线设置，所以 Compute Shader 可以通过 `_CameraDepthTexture` 读取深度。

接着创建 `Moonflow_Lensflare.mat`，Shader 选择 `Moonflow/Lensflare`，主贴图指向光晕图集。当前材质绑定的是 `Flare50mm.png`，渲染队列为透明队列 3000。由于运行时会通过 `MaterialPropertyBlock` 设置贴图，这个材质上的贴图更多是默认预览和兜底数据。

然后创建 `MFFlareAssetSlicer` 或 `MFFlareAssetCell` 资源。当前 `FlareAsset.asset` 是基于 `MFFlareAssetSlicer` 的资源，贴图为 `Flare50mm.png`，同时开启缩放淡出和透明淡出。它的 8 个元素共同构成一条光晕：不同 `offset` 让元素分布在屏幕中心线两侧，不同 `scale` 决定大小，`color` 控制各元素亮度和色调。

最后把 `MFLensFlare` 挂到 `Main Camera`，绑定 `Moonflow_Lensflare.mat`、`LightOcclusionChecker.compute` 和合适的 `fadeoutTime`。把 `MFFlareLauncher` 挂到需要发出光晕的 Light 上，指定 `asset`，如果是 Directional Light 就开启 `directionalLight`。运行后，Launcher 注册光源，主相机每帧刷新状态并绘制光晕。

## 当前实现的可维护点

这套实现已经能完整跑通自定义 Lens Flare，但有几个地方适合后续整理。`ComputeBuffer.GetData` 会让 CPU 等 GPU 结果，光源数量增多时可能产生同步开销；如果想进一步优化，可以把可见性判断也放在 GPU 路径里，或降低检测频率。遮挡槽位当前固定为 8 个，适合示例和少量强光源；生产环境可以改成动态容量或按距离、亮度筛选最重要的光源。

源码里部分中文注释存在编码异常，运行逻辑不受影响，但阅读体验会受损。后续可以统一把脚本保存为 UTF-8，并清理重复注释。`MFFlareCellAssetEditor.GetFlareRect` 的非正方形网格行号计算，以及 `MFFlareFreeAssetEditor` 的同名贴图搜索，也适合在继续扩展工具链时一起修正。

到这里，一条从图集数据、光源注册、屏幕空间定位、深度遮挡、淡入淡出、动态 Mesh 到透明叠加 Shader 的镜头光晕系统就完整闭环了。当前项目的实现把渲染逻辑集中在主相机，把美术参数保存在 ScriptableObject，把图集编辑留给自定义 Inspector，因此扩展新光晕样式时主要是在资产层调数据，而不是重写运行时代码。
